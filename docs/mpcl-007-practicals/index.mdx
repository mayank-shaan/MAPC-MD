# MPCL-007: Practicals in Psychology

## Course Overview

This practical course provides hands-on experience in conducting psychological experiments, administering tests, and applying psychological techniques.

## Course Structure

### Practical 1: Cognitive Experiments
- Experiment 1: Attention & Stroop Effect
- Experiment 2: Memory Span
- Experiment 3: Problem Solving
- Experiment 4: Decision Making

### Practical 2: Psychological Testing
- Test 1: Intelligence Assessment
- Test 2: Personality Inventories
- Test 3: Aptitude Tests
- Test 4: Interest Inventories

### Practical 3: Clinical Techniques
- Technique 1: Case History Taking
- Technique 2: Mental Status Examination
- Technique 3: Behavioral Assessment
- Technique 4: Report Writing

### Practical 4: Research Skills
- Skill 1: Data Collection
- Skill 2: Statistical Analysis
- Skill 3: Interpretation
- Skill 4: Presentation
