# MPC-003: Personality Theories and Assessment

## Course Overview

This course provides a comprehensive understanding of major personality theories and assessment methods. It explores how personality develops, is measured, and influences behavior.

## Course Structure

### Block 1: Foundations of Personality
- Unit 1: Introduction to Personality Psychology
- Unit 2: Psychoanalytic Theories (Freud)
- Unit 3: Neo-Freudian Approaches
- Unit 4: Trait Theories

### Block 2: Contemporary Theories
- Unit 1: Humanistic Theories (Rogers, Maslow)
- Unit 2: Behavioral & Social Learning Theories
- Unit 3: Cognitive Approaches
- Unit 4: Biological Perspectives

### Block 3: Personality Assessment
- Unit 1: Principles of Assessment
- Unit 2: Projective Techniques
- Unit 3: Self-Report Inventories
- Unit 4: Behavioral Assessment

### Block 4: Applications
- Unit 1: Personality & Health
- Unit 2: Personality Disorders
- Unit 3: Cross-Cultural Perspectives
- Unit 4: Personality in Applied Settings
