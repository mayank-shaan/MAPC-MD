# MPC-006: Statistics in Psychology

## Course Overview

This course covers statistical methods essential for psychological research. It includes descriptive statistics, inferential statistics, and data analysis techniques.

## Course Structure

### Block 1: Descriptive Statistics
- Unit 1: Introduction to Statistics
- Unit 2: Measures of Central Tendency
- Unit 3: Measures of Variability
- Unit 4: Data Visualization

### Block 2: Probability & Distributions
- Unit 1: Probability Concepts
- Unit 2: Normal Distribution
- Unit 3: Sampling Distributions
- Unit 4: Hypothesis Testing Basics

### Block 3: Inferential Statistics
- Unit 1: t-Tests
- Unit 2: Analysis of Variance (ANOVA)
- Unit 3: Correlation
- Unit 4: Regression Analysis

### Block 4: Advanced Topics
- Unit 1: Chi-Square Tests
- Unit 2: Non-Parametric Tests
- Unit 3: Factor Analysis
- Unit 4: Using Statistical Software
