# MPC-001: Cognitive Psychology, Learning and Memory

## Course Overview

This course provides a comprehensive understanding of cognitive processes, learning theories, and memory systems. It explores how humans perceive, process, store, and retrieve information.

## Course Structure

### Block 1: Foundations of Cognitive Psychology
- Unit 1: Introduction to Cognitive Psychology
- Unit 2: Attention and Consciousness
- Unit 3: Perception and Pattern Recognition
- Unit 4: Cognitive Neuroscience Basics

### Block 2: Memory Systems
- Unit 1: Memory Models and Theories
- Unit 2: Working Memory
- Unit 3: Long-term Memory Systems
- Unit 4: Forgetting and Memory Distortions

### Block 3: Learning Theories
- Unit 1: Classical Conditioning
- Unit 2: Operant Conditioning
- Unit 3: Observational Learning
- Unit 4: Cognitive Approaches to Learning

### Block 4: Advanced Cognitive Processes
- Unit 1: Language and Thought
- Unit 2: Problem Solving and Creativity
- Unit 3: Decision Making and Reasoning
- Unit 4: Metacognition and Cognitive Control
